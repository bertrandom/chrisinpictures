chrisinpictures
===============

Photoshops Chris into photos posted by @HistoryInPics

## Installation

Create config.json:

```
{
	"consumer_key":         "xxx",
	"consumer_secret":      "xxx",
	"access_token":         "xxx-xxx",
	"access_token_secret":  "xxx"
}
```

Install imagemagick.

Run node bot.js.
